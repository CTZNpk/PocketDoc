

1. **User Model**
   - Fields:
     - `id`: String, unique, required
     - `username`: String, required
     - `email`: String, required, unique
     - `password`: String, required
     - `createdAt`: Date, default: current date
     - 
2. **Document Model**
   - Fields:
     - `id`: String, unique, required
     - `title`: String, required
     - `content`: Text, required
     - `uploadedByUserId`: String (storing the user ID who uploaded the document)
     - `uploadDate`: Date, default: current date
     - `summary`: Text
     - `processed`: Boolean, default: false

3. **Summary Model**
   - Fields:
     - `id`: String, unique, required
     - `documentId`: String (storing the associated document ID)
     - `content`: Text, required
     - `createdDate`: Date, default: current date
     - `queryBased`: Boolean, default: false

4. **Quiz Model**
   - Fields:
     - `id`: String, unique, required
     - `summaryId`: String (storing the associated summary ID)
     - `questions`: Array of objects with fields: `question`, `options`, `correctAnswer`
     - `difficulty`: Enum (e.g., Easy, Medium, Hard)
     - `createdDate`: Date, default: current date

5. **QuizResult Model**
   - Fields:
     - `id`: String, unique, required
     - `quizId`: String (storing the associated quiz ID)
     - `userId`: String (storing the user ID who took the quiz)
     - `score`: Number
     - `completedAt`: Date

6. **Feedback Model**
   - Fields:
     - `id`: String, unique, required
     - `userId`: String (storing the user ID who submitted the feedback)
     - `content`: Text
     - `rating`: Number (e.g., 1-5)
     - `submittedAt`: Date
